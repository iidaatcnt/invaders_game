# Invaders Game

This is a classic arcade-style space invaders game implemented using HTML, CSS, and JavaScript.

## How to Play

-   **Objective:** Control your spaceship, shoot down incoming alien invaders, and prevent them from reaching the bottom of the screen.
-   **Move:** Use the **left and right arrow keys** to move your spaceship horizontally.
-   **Shoot:** Press the **space bar** to fire bullets at the invaders.

## Rules

-   Invaders move horizontally and descend one step when they hit the edge of the screen.
-   If any invader reaches the bottom of the screen (player's level), the game is over.
-   Destroy all invaders to clear the wave. The game continues with new waves of invaders.
-   The game ends if you are hit by an invader's bullet or if an invader reaches your position.
-   Press the **R key** to restart the game at any time.

## Features

-   **Classic Arcade Gameplay:** Experience the nostalgic feel of a retro space invaders game.
-   **Dynamic Invaders:** Invaders move in patterns and descend, increasing the challenge.
-   **Scoring System:** Earn points for each invader destroyed.
-   **Sound Effects:** Includes sound effects for shooting, hitting invaders, and game over.
-   **Simple Graphics:** Retro-style pixel art graphics with a bright and cheerful theme.

## How to Play Locally

1.  Clone this repository to your local machine.
2.  Open the `index.html` file in your web browser.

That's it! You can start playing right away.
